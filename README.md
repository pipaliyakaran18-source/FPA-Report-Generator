# FP&A Report Generator 📊

A Python tool that automates financial variance commentary and generates 
professional Word documents — reducing 3–4 hours of manual FP&A work to 60 seconds.

## What It Does
- Takes Actual vs Budget financial data (8 line items) as input
- Calculates variances, margins, and ratios automatically
- Generates a 5-paragraph management commentary in formal prose
- Outputs a formatted Word document with cover page, colour-coded table, and appendix

## Sample Output
See Apple_Inc_Q1_FY2026_Variance_Report.docx in the output/ folder

## How to Run
pip install -r requirements.txt
python sample_data/sample_run.py   # instant demo
python main.py                      # enter your own data
